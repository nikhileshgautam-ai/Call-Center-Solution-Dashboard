# 📊 Call Center Performance Dashboard (Power BI)

## 📌 Problem Statement
Call centers handle high volumes of customer interactions, making it challenging to monitor performance, maintain service quality, and optimize agent efficiency.

This dashboard provides a comprehensive overview of call center operations to enable data-driven decision-making.

---

## 🎯 Objectives
- Track call handling and resolution performance  
- Analyze agent efficiency and productivity  
- Monitor customer satisfaction trends  
- Identify monthly call patterns and workload distribution  

---

## ⚙️ Tools & Technologies
- Power BI  
- DAX  
- Excel Dataset  

---

## 📊 Key Metrics (KPIs)
- Total Calls Answered  
- Total Calls Resolved  
- Average Speed of Answer (65.33 sec)  
- Customer Satisfaction Score (~3.37 / 5)  

---

## 📈 Key Insights
- 📌 Over 80% of calls are successfully answered, indicating strong responsiveness  
- 📌 Resolution rate is slightly lower (~73%), highlighting improvement opportunities  
- 📌 Average response time (~65 sec) suggests moderate wait time for customers  
- 📌 Customer satisfaction score (~3.37) indicates scope for service quality improvement  
- 📌 Call volume remains relatively consistent across months with slight variation  

---

## 📊 Dashboard Features
- 🎯 **Interactive Filters:** Agent, Topic, and Date range  
- 📞 **Call Analysis:** Answered vs Unanswered calls  
- ✅ **Resolution Tracking:** Resolved vs unresolved calls  
- ⏱️ **Performance Metric:** Average speed of answer  
- 👨‍💼 **Agent Performance Table:**  
  - Calls handled  
  - Resolution count  
  - Satisfaction score  
  - Response time  
- 📅 **Monthly Trends:** Call distribution over time
